What is a ShopFi Node?

ShopFi Global is a decentralized protocol consisting of a series of components, including smart contracts, clients and nodes, and other infrastructure and services.

A node refers to a server running ShopFi Global node program after the corresponding $ShopFi is pledged. The server is responsible for executing various work tasks specified by ShopFi and obtaining corresponding incentives through proof of procedures (PoP)

How it works?

Read the flow chart below to understand the workflow of how to become a node maintainer and what role a node plays in ShopFi ecosystem.Get ready to become a node maintainer? Click Apply button to get started.

Differences between traditional PoW and ShopFi node's PoP

Unlike traditional blockchain's PoW (Proof of work) such as Bitcoin or Ethereum, ShopFi's PoP (Proof of Procedures) mechanism is environment friendly. It doesn't require a lot of power consummation. The work a node is required to perform is not based on hashing rates and puzzle resolving. Nodes are only required to finish certain series of tasks of normal work which contributes to ShopFi's requirement to keep the network up and running like automatic jobs and message pushing, etc. No need to worry about energy consuming, ShopFi values environment as all of you do.
